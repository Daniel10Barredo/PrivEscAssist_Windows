Impersonate Privilege

◍ Explicación

Esta directiva que suele estar asignada a cuentas de servicio, permite suplantar a un usuario u otra cuenta específica y actuar en su nombre, más concretamente que cuentas de usuario puede llamar a CreateProcessAsUser() de la API de Windows para que un servicio pueda iniciar a otro.

◍ Ataque

Para la explotación de este tipo de privilegios, son muy conocidos en la comunidad los ataques tipo Juicy Potato y sus variantes. Estos ataques se basan en la combinación de una suplantación de identidad NBNS y al mismo tiempo una retransmisión NTLM de forma interna para crear un servicio con las credenciales que ejecute un comando definido por el usuario. Ha habido algunos cambios por parte de Microsoft tratando de bloquear este tipo de ataques, aunque por compatibilidad todavía hay variantes que funcionan a día de hoy.

Juicy Potato Windows 7,8.1,10(<1809),2008,2012,2016 CLSID
Rotten Potato Windows 7,8.1,10(<1809),2008,2012,2016 CLSID
Print Spoofer Windows 10,11,2016,2019,2022 (No parcheado todavía)

◍ Herramientas

JuicyPotato.exe JuicyPotato32.exe RougePotato.exe PrintSpoofer64.exe PrintSpoofer32.exe
