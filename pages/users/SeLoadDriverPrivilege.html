Load Driver Privilege

◍ Explicación

Esta directiva determina qué usuarios pueden cargar y descargar controladores de dispositivo dinámicamente y estos luego van a ser ejecutados con privilegios elevados. Generalmente los grupos de operadores de impresora tienen este privilegio, sin embargo para que se pueda instalar un controlador necesita estar firmado.

◍ Ataque

Un atacante como está limitado a la firma de Windows, puede abusar de drivers vulnerables conocidos como capcom.sys que puede ser abusado para ejecutar cualquier tipo de código y además está firmado por Windows. Una vez cargado solo habría que tirar el exploit correspondiente (para capcom ) y se conseguiría ejecución como SYSTEM.
