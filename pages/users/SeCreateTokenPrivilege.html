Create Token Privilege

◍ Explicación

Esta directiva determina que usuario tiene permisos para poder crear un token y que cuentas puede usar para obtener acceso a los recursos locales usando NtCreateToken() de la API de Windows.

◍ Ataque

Esta directiva puede ser utilizada por un atacante para realizar una elevación de privilegios, ya que es posible impersonar un token siempre que sea con el mismo usuario y el nivel de integridad sea igual o menor al que se está ejecutando. En este caso, el usuario puede crear e impersonar un token añadiendo a este usuario a un grupo privilegiado que le permita la escalada por otras vías.
