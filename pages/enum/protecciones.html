Protecciones del sistema

Firewall

Comprobación del estado y la configuración del firewall.

netsh firewall show state
netsh firewall show config

Antivirus

Nombre del antivirus que está utilizando.

WMIC /Node:localhost /Namespace:\\root\SecurityCenter2 Path AntiVirusProduct Get displayName

Exclusiones de Windows Defender.

Get-ChildItem 'registry::HKLM\SOFTWARE\Microsoft\Windows Defender\Exclusions' -ErrorAction SilentlyContinue

AppLocker

Comprobar si hay restricciones de AppLocker.

Get-AppLockerPolicy -Effective | select -ExpandProperty RuleCollections
