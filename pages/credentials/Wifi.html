Credenciales Wifi

En esta sección se comprobarán las contraseñas de las diferentes redes Wifi almacenadas en el sistema. No es una práctica común reutilizar estas contraseñas, pero resultan siempre interesantes para un atacante.

Redes Wifi

Comandos para la extracción de las credenciales Wifi almacenadas en el sistema. 

netsh wlan show profiles | findstr ":" | ForEach-Object { netsh wlan show profiles key=clear name= $_.split(":")[1].trim() }

#Para listar las guardadas
netsh wlan show profiles
#Para ver las claves de cada una
netsh wlan show profiles key=clear name=[Nombre_Wifi]
