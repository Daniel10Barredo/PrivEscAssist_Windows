Credenciales en Entorno

En ocasiones, algunas aplicaciones usan las variables de entorno para guardar sus credenciales y poder funcionar de forma autónoma, es por eso que resulta interesante revisar siempre el entorno en búsqueda de estas.

Variables de Entorno

Comandos para buscar posibles contraseñas en variables de entorno. 

#Para buscar por patrón
Get-ChildItem Env: | findstr /i "passw"
#Para enumerar todas
Get-ChildItem Env:

#Para buscar por patrón
set | findstr /i "passw"
#Para enumerar todas
set
