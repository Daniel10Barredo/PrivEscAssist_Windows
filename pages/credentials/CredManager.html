Credential Manager

El administrador de credenciales de Windows permite a los usuarios guardar información de login de webs, aplicaciones, redes, etc. En esta sección se comprobarán si existen credenciales dentro del administrador que puedan ser utilizadas para la escalada de privilegios.

◍ Comprobación

Se pueden listar las credenciales con el siguiente comando para averiguar si hay credenciales almacenadas. 

cmdkey /list

◍ Ataque

Aunque se pueda ver que las credenciales estan almacenadas, sin privilegios elevados no se pueden extraer solamente se pueden utilizar por ejemplo si son credenciales de otro usuario con el comando runas. 

runas /savecred /user:[user] "C:\evil.exe"
